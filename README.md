A standard software project.

## Installation

1.  Clone the repository: `git clone https://github.com/seuroproject/curso-web-development.git`
2.  Navigate to the project directory: `cd curso-web-development`
3.  Install dependencies: `npm install` (or equivalent for your chosen package manager).
4.  Run the development server: `npm start` (or equivalent).
